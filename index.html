Features

더 안전하고, 더 편리하게.

유키노트는 비밀번호를 외우는 대신 떠올리는 새로운 방식으로, 보안과 편의성을 동시에 제공합니다.

🔑

키워드 기반 비밀번호 생성

나만의 추억이나 키워드 두 개만 입력하면 강력한 비밀번호를 즉시 생성. 같은 키워드로 언제든 동일한 비밀번호를 재생성할 수 있습니다.

🎲

랜덤 비밀번호 생성

급하게 비밀번호가 필요할 때 길이와 옵션을 설정하여 랜덤 비밀번호를 즉시 만들어 사용하세요.

📋

스마트 계정 관리

사이트별 아이디, 비밀번호, 메모를 체계적으로 관리. 초성 검색과 그룹 분류로 빠르게 찾을 수 있습니다.

📊

보안 대시보드

내 비밀번호의 건강 점수를 한눈에 확인. 노후화 분포와 UkiNote 엔진 사용률까지 체크합니다.

⚡

미니 금고 위젯

앱을 열지 않아도 다른 앱 위에 떠 있는 플로팅 위젯으로 ID/PW를 바로 복사할 수 있습니다.

🛡️

100% 오프라인 · 이중 보안

모든 데이터는 기기 내부에만 저장. 생체인식 잠금과 백업 암호로 이중 보호합니다.
